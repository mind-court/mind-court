# Security Policy

## Reporting a vulnerability

If you've found a security issue in Mind Court — anything from a bug that could expose user data to a misconfiguration in this repository — please **do not** open a public issue.

Instead, email **bensmith0124@gmail.com** with:

- A description of the issue
- Steps to reproduce (if applicable)
- The potential impact as you see it

I'll acknowledge receipt within **3 business days** and aim to provide a status update within **7 days**. If the issue is confirmed, I'll work with you on a coordinated disclosure timeline.

Thank you for helping keep Mind Court and its users safe.
